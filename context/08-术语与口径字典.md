# 08-术语与口径字典

> 统一本项目常用术语，避免同一对象在不同文档里叫不同名字。AI 处理本项目任务时以此为准。

## 一、主数据术语

| 术语 | 统一口径 | 禁止混用 |
| :--- | :--- | :--- |
| 仓库档案 | 仓库的主数据记录，包含仓库编码、名称、类型、负责人 | 库房信息、仓储资料 |
| 供应商档案 | 供应商的主数据记录，包含编码、名称、联系方式、结算方式 | 厂商信息、供货商 |
| 客户档案 | 客户的主数据记录，包含编码、名称、等级、价格级别、结算方式 | 买家信息、下游客户 |
| 商品档案 | 商品（SKU）的主数据记录，包含编码、名称、分类、价格 | 产品信息、货品资料 |
| 启用 | 主数据可被下游业务单据选择的状态 | 激活、上架、开启、正常 |
| 停用 | 主数据不可被新建业务单据选择的状态（历史可见） | 禁用、冻结、下架、注销 |

## 二、业务单据术语

| 术语 | 统一口径 | 备注 |
| :--- | :--- | :--- |
| 销售订单 | 批发客户下单后形成的业务订单 | 和零售单区分 |
| 销售出库 | 基于销售订单执行的实物出库动作 | 和订单分离 |
| 零售收银 | 门店现场成交与收款动作 | 按轻量收银模式处理 |
| 采购订单 | 向供应商下单的业务单据 | 采购起点 |
| 采购入库 | 供应商到货后确认入库的动作 | 可有差异（实际到货量 ≠ 订单量） |
| 采购退货 | 基于原采购入库发起的退货动作 | 优先基于入库单发起，不是基于采购订单 |
| 库存查询 | 查看现存、占用、可用库存的能力 | 共享底账视角 |
| 收款登记 | 记录客户回款的动作 | 不等于复杂核销 |
| 付款登记 | 记录向供应商付款的动作 | 当前不做复杂核销 |

## 三、价格术语

| 术语 | 统一口径 | 备注 |
| :--- | :--- | :--- |
| 默认批发价 | 商品批发开单时的基础参考价格 | 最终成交价受客户价格级别影响 |
| 默认零售价 | 商品零售收银时的默认带出价格 | 不参与零售的商品可为空 |
| 参考采购价 | 采购下单时的参考成本价格 | 仅作录单参考，不作财务结算口径 |
| 账期 | 约定的付款宽限天数 | 如"账期30天"=到货后30天内回款 |
| 信用额度 | 客户最大允许挂账金额上限 | 一期不做自动强控，仅作参考提醒 |

## 四、操作动词规范

| 规范用词 | 含义 | 禁止混用 |
| :--- | :--- | :--- |
| 新增 | 创建新主数据或新单据 | 新建、创建、添加 |
| 编辑 | 修改已有记录 | 修改、更新、变更 |
| 查看 | 进入详情页只读展示 | 详情、预览 |
| 停用 | 终止主数据可被新单引用 | 禁用、冻结、下架 |
| 启用 | 恢复主数据可被新单引用 | 激活、解冻、上架 |
| 取消 | 撤回单据（下游全部未执行时） | 作废 |
| 搜索 | 提交查询条件、按条件筛选 | 查询（作为按钮文案时禁用） |
| 导出 | 导出当前列表或筛选结果 | 下载、导出 Excel |

## 五、库存口径术语

| 术语 | 含义 |
| :--- | :--- |
| 现存 | 仓库里实际有的数量 |
| 占用 | 已被销售订单锁定但未出库的数量 |
| 可用 | 现存 - 占用 = 还可以承诺给新订单的数量 |

## 六、数量口径术语（采购专用）

| 术语 | 含义 | 谁填 |
| :--- | :--- | :--- |
| 采购数量 | 采购订单上计划购买的数量 | 采购员（订单上） |
| 实收数量 | 供应商送货到仓后，仓库实际收到的数量 | 仓管（入库时） |
| 入库数量 | 确认入库计入库存的数量 | 系统（仓管确认后） |

> 关键规则：入库数量 ≤ 实收数量 ≤ 采购订单的未入库数量。三个数量不一定相等。
