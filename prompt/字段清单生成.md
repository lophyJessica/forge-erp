# 字段清单生成

> **适用场景**：单独生成或更新某个单据的字段清单。
> **前置准备**：对应的主 PRD + templates/字段清单模板.md + context/05-单据分层与编码规则.md

---

## 提示词

```
请为「[单据名称]」生成字段清单，格式严格遵循 templates/字段清单模板.md。

【输入】
- prd-docs/[路径]/[单据名]主PRD.md（业务背景和规则来源）
- templates/字段清单模板.md（格式规范）
- context/05-单据分层与编码规则.md（命名和编码规则）

【要求】
- 头部字段、商品明细字段、统计汇总字段、系统字段四部分必须完备
- 每个字段标注：来源类型（系统生成/用户填写/系统计算/系统带出并存储/下游回写）
- 枚举值格式：`CODE=中文含义`
- 新增页/编辑页/列表/详情页的展示规则必须标注
- 继承自上游单据的字段标注"继承自 [上游单据名] 字段清单"
- 快照存储的字段标注"快照存储；[来源档案]修改后不影响已有单据数据"
- 审核后不可修改的字段在备注中标注

输出完整文件，不要省略列。
```
