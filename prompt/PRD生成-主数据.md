# PRD 生成：主数据

> **适用场景**：为基础资料模块（商品/客户/供应商/仓库）生成 PRD。
> **前置准备**：context/ + templates/PRD模板-主数据.md + templates/字段清单模板.md

---

## 提示词

```
请基于以下输入，为「[主数据名称]档案」生成主数据 PRD 文档套件：

【喂入文件】
- context/ 下全部文件
- templates/PRD模板-主数据.md
- templates/字段清单模板.md
- prd-docs/基础资料/[参考模块]/（格式参照）

【输出要求】
生成以下文件，保存到 prd-docs/基础资料/[模块名]/ 目录：

1. **[主数据名称]档案主PRD.md**
2. **[主数据名称]档案字段清单.md**
3. **[主数据名称]档案_前端Demo版PRD.md**

【关键约束】
- 主数据不提供删除，统一用"停用"替代
- 新增时默认状态为"启用"，新增表单不展示状态字段
- 停用后不可被新建业务单据选择，历史单据保留完整数据
- 文档风格与参照 PRD 保持一致
```
