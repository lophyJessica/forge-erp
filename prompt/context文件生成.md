# context 文件生成

> **适用场景**：为项目生成或补充 context/ 下的上下文文件。
> **前置准备**：已有的 context 文件 + 需要生成的目标文件骨架。

---

## 提示词 A：基于已有信息生成

```
请读取 context/ 下所有文件，然后帮我填写 context/[编号]-[文件名].md。

这个文件要回答的问题是：
- [问题1]
- [问题2]
- [问题3]

请直接输出完整文件内容。如果某些信息在已有 context 中没有覆盖，标注"[待补充]"并说明需要什么信息。
```

## 提示词 B：从空白生成新 context 文件

```
我需要为项目新建一个 context 文件：context/[编号]-[文件名].md。

这个文件的目标是：[一句话说明]

请基于以下信息生成：
- [我提供的项目信息/口述内容]

输出格式参考 context/ 中已有文件的风格——编号、章节结构、表格格式保持一致。如果信息不足，标注"[待补充]"。
```
