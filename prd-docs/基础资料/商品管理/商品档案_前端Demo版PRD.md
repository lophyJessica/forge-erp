# 商品档案_前端Demo版PRD

**version**：v1.2.0

**date**：2026-07-13

**description**：商品档案前端页面规范，统一承接商品列表页、查看页、新增页、编辑页。字段定义以《商品档案字段清单》为准；本文档只定义页面结构、控件类型、显隐逻辑、交互与 Mock 数据要求。

---

## 一、页面清单

| 页面 | 目标 |
| :--- | :--- |
| 商品列表页 | 查询商品、查看状态、进入新增/编辑/详情 |
| 商品详情页 | 查看商品基础资料和价格信息 |
| 新增商品页 | 创建新商品档案 |
| 编辑商品页 | 修改已有商品档案 |

---

## 二、列表页规范

### 2.1 查询条件区

| 字段 | 控件类型 | 说明 |
| :--- | :--- | :--- |
| 综合搜索 | Input | 支持商品编码 / 商品名称 / 条码 |
| 商品分类 | Select | 全部 / 各分类枚举值（引用字段清单） |
| 状态 | Select | 全部 / 启用 / 停用 |

### 2.2 表格列定义

| 列名 | 展示规则 | 说明 |
| :--- | :--- | :--- |
| 商品编码 | 文本 | 默认展示 |
| 商品名称 | 文本 | 默认展示 |
| 商品分类 | 文本 | 默认展示 |
| 规格型号 | 文本 | 默认展示 |
| 销售单位 | 文本 | 默认展示 |
| 安全库存 | 整数 | 展示 `safetyStock`；未维护时展示 `-` |
| 默认批发价 | 金额 | 保留2位小数，千分位格式 |
| 默认零售价 | 金额 | 保留2位小数，千分位格式；空值展示 `-` |
| 状态 | StatusPill | 启用=绿色 / 停用=灰色 |
| 最后修改时间 | 时间 | `YYYY-MM-DD HH:mm:ss`，辅助判断数据新鲜度 |
| 操作 | 固定右侧 | 查看 / 编辑 / 停用（启用态）/ 启用（停用态） |

> **操作列说明**：启停用通过行操作按钮触发确认弹窗，不在主表单中操作。停用弹窗内含"停用原因"必填输入框；启用弹窗为纯确认。

## 三、查看页规范

### 3.1 页面操作区

| 按钮 | 展示条件 | 说明 |
| :--- | :--- | :--- |
| 返回列表 | 始终显示 | 返回商品列表页 |
| 编辑 | 始终显示 | 进入编辑页 |
| 停用 | 当前状态=启用 | 触发停用确认弹窗 |
| 启用 | 当前状态=停用 | 触发启用确认弹窗 |

### 3.2 信息分组

| 分组 | 字段 |
| :--- | :--- |
| 基础资料 | 商品编码、商品名称、商品分类、规格型号、销售单位、条码 |
| 价格与说明 | 默认零售价、默认批发价、参考采购价、安全库存、商品备注 |
| 补充信息 | 停用原因 |
| 系统信息 | 创建人、创建时间、最后修改人、最后修改时间 |

> **空值展示**：所有字段无数据时统一展示 `-`，不留空白。  
> **停用原因**：仅当商品状态为"停用"时，在"补充信息"分组中展示实际停用原因；启用状态下展示 `-`。

## 四、新增 / 编辑页规范

### 4.1 字段控件与显隐规则

| 字段 | 控件 | 必填 | 显隐条件 |
| :--- | :--- | :--- | :--- |
| 商品编码 | Input | 是 | 新增显示；编辑已引用时只读 |
| 商品名称 | Input | 是 | 始终显示 |
| 商品分类 | Select | 是 | 始终显示 |
| 规格型号 | Input | 是 | 始终显示 |
| 销售单位 | Select | 是 | 始终显示 |
| 条码 | Input | 否 | 始终显示 |
| 默认零售价 | Input（≥0，保留2位小数） | 否 | 始终显示 |
| 默认批发价 | Input（≥0，保留2位小数） | 是 | 始终显示 |
| 参考采购价 | Input（≥0，保留2位小数） | 否 | 始终显示 |
| 安全库存 | InputNumber（≥0，整数） | 否 | 始终显示 |
| 商品备注 | TextArea | 否 | 始终显示 |
| 停用原因 | TextArea | 条件必填 | **仅在停用弹窗中展示**，不在主表单中出现 |

> **说明**：启停用状态不在新增/编辑主表单中以字段形式出现。启/停用操作通过列表页行操作按钮或详情页操作按钮触发确认弹窗完成，弹窗内含停用原因输入框（停用时必填）。

### 4.2 模式差异说明

| 比较维度 | 新增模式 | 编辑模式 | 查看模式 |
| :--- | :--- | :--- | :--- |
| 页面标题 | 新增商品 | 编辑商品 | 商品详情 |
| 商品编码 | 可编辑 | 已引用时只读 | 纯文本 |
| 启停用状态 | 默认启用，新增时不展示状态字段 | 不在主表单展示，通过页面操作按钮触发 | 展示当前状态（StatusPill） |
| 停用原因 | 不展示 | 不在主表单展示（通过弹窗输入） | 停用状态下展示实际原因，启用状态展示 `-` |
| 系统信息 | 不展示 | 展示（只读） | 展示（只读） |

## 五、输入长度与格式限制

| 字段 | 长度限制 | 字符限制 | 补充说明 |
| :--- | :--- | :--- | :--- |
| 商品编码 | 最多 30 个字符 | **仅允许**大写字母（A-Z）、数字（0-9）、连字符（-）；**禁止**中文、空格及其他特殊字符 | 参考格式 `SKU-0001`；最短 2 个字符 |
| 商品名称 | 最多 50 个字符 | 禁止纯空白，允许中文和常规符号 | 保存前自动去首尾空格 |
| 商品分类 | 固定枚举 | 从枚举值中选择 | 一期按单级分类处理 |
| 规格型号 | 最多 100 个字符 | 允许中文、字母、数字和常规规格符号 | 如 `256GB/深空灰色` |
| 销售单位 | 固定枚举 | 件 / 盒 / 套 / 包 | 必填 |
| 条码 | 最多 30 个字符 | 数字或字母数字组合 | 全局唯一；选填 |
| 默认零售价 | - | ≥ 0，保留 2 位小数 | 选填 |
| 默认批发价 | - | ≥ 0，保留 2 位小数 | 必填 |
| 参考采购价 | - | ≥ 0，保留 2 位小数 | 选填 |
| 安全库存 | - | 非负整数 | 选填；未维护时为空，不参与低库存预警展示 |
| 商品备注 | 最多 200 个字符 | 普通备注文本，不做特殊字符硬限制 | 选填 |
| 停用原因 | 最多 100 个字符 | 普通备注文本，不做特殊字符硬限制 | 停用时弹窗内必填 |

## 六、校验与保存规则

### 6.1 校验规则

| 校验点 | 规则 | 时机 |
| :--- | :--- | :--- |
| 商品编码（必填/唯一） | 必填；最多 30 个字符；全局唯一 | 保存时全量校验 |
| 商品编码（格式） | 仅允许大写字母（A-Z）、数字（0-9）、连字符（-）；不允许中文、空格及其他字符；输入非法字符时**立即阻断**，不允许录入 | 输入时实时阻断 + 保存时全量校验 |
| 商品名称 | 必填，最长 50 个字符 | 保存时全量校验 |
| 商品分类 | 必填 | 保存时全量校验 |
| 规格型号 | 必填，最长 100 个字符 | 保存时全量校验 |
| 销售单位 | 必填 | 保存时全量校验 |
| 默认批发价 | 必填，≥ 0，保留 2 位小数 | 保存时全量校验 |
| 条码 | 选填；填写时必须全局唯一 | 失焦或保存时校验 |
| 默认零售价 / 参考采购价 | 选填；填写时 ≥ 0，保留 2 位小数 | 保存时校验 |
| 安全库存 | 选填；填写时必须为非负整数 | 保存时校验 |
| 停用原因（弹窗） | 停用动作触发时必填，最长 100 个字符 | 弹窗确认时校验 |

补充说明：

- 编辑模式下，已被业务单据引用的商品，编码字段以只读输入框或纯文本方式展示，不允许修改，不触发唯一性校验
- 价格字段填写 0 视为合法（如内赠商品）

### 6.2 保存规则

| 场景 | 规则 |
| :--- | :--- |
| 新增保存成功 | 返回列表页，并在列表中看到最新新增数据 |
| 编辑保存成功 | 返回列表页，并在列表中看到最新编辑结果 |
| 保存失败 | 阻断提交，定位首个报错字段 |
| 编码已引用 | 不允许修改编码，不触发唯一性校验，直接以只读展示 |

## 七、反馈规范

| 操作 | 反馈方式 | 文案 |
| :--- | :--- | :--- |
| 保存成功 | Toast | 保存成功 |
| 启用成功 | Toast | 商品已启用 |
| 停用成功 | Toast | 商品已停用 |
| 编码格式非法 | 字段红字提示（实时） | 商品编码只允许大写字母、数字和连字符（-） |
| 编码重复 | 字段红字提示 | 商品编码已存在 |
| 条码重复 | 字段红字提示 | 商品条码已存在 |
| 安全库存格式非法 | 字段红字提示 | 安全库存必须为非负整数 |
| 引用后改编码 | Toast | 该商品已被业务单据引用，编码不可修改 |
| 停用时未填写原因 | 弹窗内字段红字提示 | 停用原因不能为空 |

## 八、验收重点

- [ ] 列表默认列覆盖商品识别、价格和状态信息
- [ ] 商品编码在已引用后不可编辑
- [ ] 启停用通过操作按钮弹窗触发，不在主表单中出现状态字段
- [ ] 弹窗内停用原因必填校验正确
- [ ] 停用商品不能继续在新单中选择
- [ ] 空值字段统一展示 `-`，不留空白
- [ ] 安全库存支持选填，仅接受非负整数，并在列表页和详情页正确展示
- [ ] 停用原因仅在商品状态为"停用"时展示实际内容
- [ ] 新增保存成功后返回列表页，并可看到最新新增数据
- [ ] 编辑保存成功后返回列表页，并可看到最新编辑结果
- [ ] 价格字段填写 0 视为合法，不应报错
