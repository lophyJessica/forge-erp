# 库存流水查询字段清单

> **文档定位**：库存流水查询页面所有字段的唯一定义来源。下游文档只引用字段名称，不重复定义字段含义及取值范围。
>
> **引用规则**：
> - 本模块为只读查询台账，字段清单中剔除“新增页”和“编辑页”列，改为「查询条件」、「列表展示」及「详情展示」
> - 下游文档（主PRD、Demo PRD）只引用本文件定义的字段名，严禁重复定义字段本体
>
> **版本**：V1.0 | 更新时间：2026-07-04

## 一、查询条件字段

| 字段名称 | 控件类型 | 取值说明 | 必填性 | 默认值 | 过滤匹配规则 | 说明 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 仓库 | `SelectWithSearch` | 来源于启用中的仓库档案；展示格式：`{Code} {Name}` | 选填 | 空（全部仓库） | 精确匹配 | 筛选特定仓库商品变动 |
| 商品 | `SelectWithSearch` | 来源于启用中的商品档案；支持编码或名称联想检索 | 选填 | 空（全部商品） | 精确匹配 | 筛选特定商品变动 |
| 单据类型 | `Select` | 枚举：`PI=采购入库` / `PRO=采购退货出库` / `SOO=销售出库` / `SR=销售退货入库` / `RS=零售单成交` / `TR_OUT=调拨出库` / `TR_IN=调拨入库` / `BL=报损出库` / `CK_IN=盘盈入库` / `CK_OUT=盘亏出库` | 选填 | 空（全部类型） | 精确匹配 | 筛选具体业务动作引起的流水 |
| 日期范围 | `DateRange` | 格式：`YYYY-MM-DD` 到 `YYYY-MM-DD` | 选填 | 空（不限） | 区间匹配 | 筛选时间段内的流水；最大查询跨度为 365 天 |
| 变动方向 | `Select` | 枚举：`IN=入库（+）` / `OUT=出库（-）` | 选填 | 空（全部方向） | 精确匹配 | 筛选库存增加或减少的单向流 |
| 批次号 | `Input` | 文本输入 | 选填 | 空 | 模糊匹配 | 筛选特定批次实物库存的变动流水 |
| 来源单号 | `Input` | 文本输入 | 选填 | 空 | 模糊匹配 | 精确/模糊输入单号检索（如 PO/PI/SOO/RS/TR 等） |

## 二、列表展示字段

| 字段名称 | 字段来源 | 取值说明 | 列表展示 | 建议列宽 | 详情展示 | 字段说明 | 备注 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 库存流水号 | 系统生成 | 格式：`FL{YYYYMMDD}-{8位序号}`，如 `FL20260704-00000001` | 是 | 180px | 是 | 每一笔库存物理变动的系统唯一审计标识 | 只读主键，全局唯一，不可回收 |

| 规格型号 | 系统带出 | 来源于商品档案的规格型号 | 是 | 130px | 是 | 商品规格快照，无规格展示 `-` | 快照存储 |
| 单位 | 系统带出 | 来源于商品档案的计量单位 | 是 | 80px | 是 | 商品计量单位快照 | 快照存储 |
| 变动类型 | 系统带出 | 根据来源单据类型自动归类映射（如采购入库、销售出库等） | 是 | 120px | 是 | 标识具体的库存业务类型 | 对应一、查询条件中的“单据类型” |
| 变动数量 | 系统生成 | 整数，入库业务为正数（带 `+`），出库业务为负数（带 `-`） | 是 | 110px | 是 | 本次实物入库或出库的件数 | 不展示千分位 |

| 操作人 | 系统生成 | 当前登录用户姓名 | 是 | 110px | 是 | 触发并执行该笔单据业务的操作人员 | - |

## 三、系统字段

库存流水属于**只读审计日志**，系统字段极其简化，不包含任何修改或审核人员信息。

| 字段名称 | 字段来源 | 取值说明 | 必填性 | 展示说明 | 字段说明 | 备注 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| 创建人 | 系统生成 | 触发变动单据的最后提交/确认人 | 系统自动 | 详情页只读 | 记录最初的流水创建者 | - |
| 创建时间 | 系统生成 | `YYYY-MM-DD HH:mm:ss` | 系统自动 | 同“发生时间” | 流水落库记录时间 | 与发生时间一致 |

---

## 四、展示与展示规则

- **千分位及符号**：变动数量须加带正负号（入库带 `+` 且绿色字体，出库带 `-` 且红色/黑色字体）。变动后现存量仅作整数展示。
- **列表默认排序**：默认按 `发生时间` 降序（DESC）展示。
- **关联跳转**：列表中「来源单号」及「库存流水号」为可点击高亮格式，点击可在浏览器新页签打开该关联单据的详情页。
