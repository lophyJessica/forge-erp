# 采购入库单字段清单

> **文档定位**：采购入库单所有字段的唯一定义来源。下游文档只引用字段名称，不重复定义枚举值、必填性、计算规则。
>
> **引用规则**：
> - 采购入库单继承采购订单字段时，标注「继承自采购订单」，以保障字段本体的单一事实源（SSOT）
> - 下游文档（Demo PRD、主PRD）只引用字段名称，严禁在其他位置重复定义字段本体
>
> **修改原则**：字段定义变化 → **只改本文件**；下游 Demo PRD 不手动修改，由 AI 重新生成。
>
> **版本**：V1.0 | 更新时间：2026-07-04

## 五、展示与引用规则

- 采购入库单被下游单据（如采购退货单）引用时，展示格式为采购入库单号（如 `PI20260704-0001`）
- 列表优先展示列：采购入库单号、来源采购单号、供应商、入库仓库、入库日期、入库状态、入库总金额、最后修改时间
- 采购入库单明细中的商品资料与单价均继承自采购订单，在入库时不可被用户直接修改，确保上下游价格与合同口径绝对一致
- 数量与金额字段展示格式：千分位展示（如 `¥10,000.00`），实收/入库数量仅展示整数无千分位
