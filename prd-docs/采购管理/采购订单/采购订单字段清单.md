# 采购订单字段清单

> **文档定位**：采购订单所有字段的唯一定义来源。下游文档只引用字段名称，不重复定义枚举值、必填性、计算规则。
>
> **引用规则**：
> - 采购入库单继承采购订单头部字段时，标注「继承自采购订单」，不重复写字段定义
> - 下游文档（Demo PRD、主PRD）只引用字段名称写规则，严禁重复定义字段本体
>
> **修改原则**：字段定义变化 → **只改本文件**；Demo PRD 不手动修改，由 AI 重新生成。
>
> **按需维度说明**：一期无批量导入功能规划，「可导入」列不加入本清单。
>
> **版本**：V1.1 | 更新时间：2026-04-20

## 五、展示与引用规则

- 采购订单被下游单据（如采购入库单）引用时，展示格式为采购单号（如 `PO20260419-0001`）
- 列表优先展示列：采购单号、供应商、入库仓库、下单日期、预计到货日期、订单状态、采购总金额、最后修改时间
- 已审核的采购订单，头部字段（供应商、仓库、下单日期、商品明细数量价格）均不允许修改；仅采购备注、行备注允许在待入库和部分入库状态下继续编辑
- 商品明细中「商品名称」「商品条码」「规格型号」「单位」均为商品快照，选中商品后系统自动带出并存储，后续与商品档案变化解耦
