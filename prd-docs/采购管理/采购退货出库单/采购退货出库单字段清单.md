## 文档定位

本文件是采购退货出库单（PRO）所有字段的唯一定义来源。下游 Demo PRD、业务规则和流程推演只引用字段名称，不重复定义枚举值、必填性和计算规则。

**引用规则**：
- PRO 继承采购退货单（PR）字段时，标注「继承自采购退货单」，不重复改写 PR 字段本体。
- PRO 确认出库后生成库存流水（FL）和应付冲减结果；FL 编号以前缀规则为准。
- 字段定义变化时，优先修改本文件，再同步重生成下游文档。

## 五、展示与引用规则

- PRO 被库存流水引用时，来源单据展示为采购退货出库单号，如 `PRO20260707-0001`。
- 列表优先展示列：采购退货出库单号、来源退货单号、供应商、出库仓库、出库日期、出库状态、出库总数量、出库总金额、最后修改时间。
- PRO 明细中的商品资料、单价、税率均继承自 PR，出库时不可被用户直接修改。
- 确认出库后全部字段锁定；系统同步扣减现存、冲减供应商应付，并生成库存流水 FL。
- 数量字段展示整数；金额字段展示 `¥1,234.56`。
