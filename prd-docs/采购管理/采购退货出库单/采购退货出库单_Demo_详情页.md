---

## 一、页面概述

| 项目 | 内容 |
| :--- | :--- |
| 页面名称 | 采购退货出库单详情 |
| 页面目标 | 提供对已确认、草稿及已作废采购退货出库单的完整信息查看、上游溯源和流程流转按钮 |
| 页面路径 | `/purchase/return-shipments/{id}` |
| 骨架结构 | 页头(单号+状态) → 状态区 → 基本信息卡片 → 明细清单卡片 → 关联下游卡片 → 固定底部操作栏 |

---

## 二、状态区

状态区位于页面最上方，用以反映该张单据当前生命周期节点。

### 2.1 状态 Badge

- 草稿：灰色 Badge
- 已确认：绿色 Badge
- 已作废：红色 Badge

### 2.2 状态时间线

只读展示：
- 创建时间：`创建人` 于 `创建时间` 从 PR 下推创建。
- 作废时间：仅状态为已作废时展示，使用最后修改人和最后修改时间。
- 出库生效时间：仅状态为已确认时展示，`确认人` 于 `确认时间` 确认出库，库存与应付已更新。

---

## 三、基本信息卡片

卡片标题：**基本信息**。采用只读 Key-Value 栅格布局展示，桌面端 4 列。

- 供应商：`{供应商编码} {供应商名称}`
- 出库仓库：`{仓库编码} {仓库名称}`
- 出库日期：`YYYY-MM-DD`
- 来源退货单号：`PR{日期}-{序号}`，链接格式，点击打开关联 PR 详情
- 来源入库单号：`PI{日期}-{序号}`，链接格式，点击打开关联 PI 详情
- 来源采购单号：`PO{日期}-{序号}`，链接格式，点击打开关联 PO 详情
- 退货原因：直接引用 PR 退货原因展示，无值时显示 `-`
- 出库备注：展示本单出库备注，无值时显示 `-`

---

## 四、明细清单卡片

卡片标题：**出库明细清单**。

### 4.1 表格列定义（只读）

| 列名 | 内容说明 | 宽度 |
| :--- | :--- | :--- |
| 序号 | 自动生成，从 1 起递增 | 60px |
| 商品编码 | 展示商品编码，只读 | 120px |
| 商品名称 | 商品名称快照，只读 | 200px |
| 条码 / 规格 | 商品条码与规格，只读 | 180px |
| 单位 | 商品计量单位快照 | 80px |
| PR退货数量 | PR 已确认退货数量 | 110px |
| 已出库数量 | 同一 PR 行已确认 PRO 累计出库数量 | 110px |
| 可出库数量 | 当前剩余可出库数量 | 110px |
| 出库数量 | 本单实际确认出库数量 | 100px |
| 单价（含税） | 采购含税单价，格式：`¥{价格}` | 110px |
| 金额（含税） | 本行出库含税金额，格式如 `¥26,500.00` | 130px |
| 行备注 | 本行商品备注，无值显示 `-` | 150px |

### 4.2 底部统计汇总（只读）

- 商品种数：`N 种`
- 出库总数量：`N`
- 出库总金额：`¥{总金额}`（千分位，2 位小数，加粗展示）

---

## 五、关联下游卡片

仅在 PRO 状态为已确认时展示，以实现单据执行结果追溯。

* **关联库存流水**：展示自动生成的流水号 `FL{日期}-{8位序号}`，点击可跳转库存流水台账明细。
* **应付冲减结果**：展示供应商、冲减金额、冲减后应付余额；应付记录编号前缀未在 context/05 定义，因此本 Demo 不展示自创编号。

---

## 六、固定底部操作栏

| 当前状态 | 展示的操作按钮（从左到右） | 交互行为与跳转 |
| :--- | :--- | :--- |
| 草稿 | `返回列表` (次)　`删除` (次)　`作废` (次)　`编辑` (次)　`确认出库` (主) | 删除/作废触发二次确认；编辑跳转执行出库页；确认出库触发全面校验与二次确认，成功后刷新当前页 |
| 已确认 | `返回列表` (次) | 返回列表页 |
| 已作废 | `返回列表` (次) | 返回列表页 |

---

## 七、二次确认弹窗规格

| 触发操作 | 弹窗标题 | 弹窗内容 | 确认按钮 | 取消按钮 |
| :--- | :--- | :--- | :--- | :--- |
| 删除 | 确认删除 | 删除后不可恢复，该草稿退货出库单将从系统中永久移除，确认删除？ | 确认删除 | 取消 |
| 作废 | 确认作废 | 作废后该退货出库单将永久失效，确认作废？ | 确认作废 | 取消 |
| 确认出库 | 确认出库 | 确认后将扣减现存库存、冲减供应商应付并生成库存流水，确认出库？ | 确认出库 | 取消 |

---

## 八、空值占位和展示规则

| 规则 | 说明 |
| :--- | :--- |
| 空值 | 统一展示 `-` |
| 主数据引用 | 统一展示 `{Code} {Name}` |
| 金额 | 保留 2 位小数，千分位展示 |
| 数量 | 整数展示 |
| 日期时间 | 日期 `YYYY-MM-DD`；时间戳 `YYYY-MM-DD HH:mm:ss` |
| 只读展示 | 使用只读文本，不使用 disabled 输入框 |

## 九、反馈文案

| 操作 | 反馈类型 | 文案 |
| :--- | :--- | :--- |
| 删除成功 | Toast + 返回列表 | 采购退货出库单已删除 |
| 作废成功 | Toast + 页面刷新 | 采购退货出库单已作废 |
| 确认出库成功 | Toast + 页面刷新 | 出库已确认，库存与应付账款已更新 |
| 状态不可编辑 | Toast | 当前状态不可编辑 |
