## 二、状态 Tab 栏

### 2.1 Tab 项定义

| Tab 标签 | 过滤逻辑 |
| :--- | :--- |
| 全部 | 展示所有状态的采购退货出库单（默认选中） |
| 草稿 | 仅展示草稿状态 |
| 已确认 | 仅展示已确认状态 |
| 已作废 | 仅展示已作废状态 |

### 2.2 计数规格

- 每个 Tab 标签右侧展示当前条件下的出库单数量，格式：`Tab名称 N`。
- 计数超过 99 时展示 `99+`，计数为 0 时仍展示 `0`。
- 计数反映全部查询条件叠加后的结果数。

### 2.3 交互行为

- 点击 Tab 切换后，列表数据及分页立即刷新，页码重置为第 1 页。
- 切换到具体状态 Tab 时，查询区「出库状态」筛选条件隐藏；切换回「全部」时恢复。
- Tab 切换不清空其他查询条件。

## 三、查询区

### 3.1 布局说明

- 桌面端 6 列栅格，默认展示 1 行；展开后展示第 2 行。
- 右侧操作组：`展开/收起` → `重置` → `搜索`。

### 3.2 查询条件

| 字段 | 控件类型 | 规格说明 |
| :--- | :--- | :--- |
| 采购退货出库单号 | `BatchSearchInput` | 支持换行或英文逗号分隔多值，精确匹配；最多 1000 行 |
| 来源退货单号 | `BatchSearchInput` | 支持多 PR 单号精确匹配 |
| 来源入库单号 | `BatchSearchInput` | 支持多 PI 单号精确匹配；展开后展示 |
| 供应商 | `SelectWithSearch` | 展示 `{Code} {Name}`；支持编码或名称搜索；可清空 |
| 出库仓库 | `SelectWithSearch` | 展示 `{Code} {Name}`；支持编码或名称搜索；可清空 |
| 出库状态 | `Select` | 枚举见字段清单；仅在「全部」Tab 下展示 |
| 出库日期 | `DateRange` | 精确到日；可只选一端；含「今天」「本周」「本月」快捷选项 |
| 最后修改时间 | `DateRange` | 展开后展示 |

## 四、工具条

工具条位于查询卡片与表格之间。由于 PRO 必须由已确认 PR 下推生成，因此工具条不提供普通新增按钮，仅保留导出操作。

| 按钮 | 位置 | 状态 | 说明 |
| :--- | :--- | :--- | :--- |
| 导出 | 左侧 | 常驻 | 导出当前查询结果；有勾选时导出选中单据 |

## 五、表格

### 5.1 表格行为

| 行为 | 规格 |
| :--- | :--- |
| 行勾选 | 支持，全选/单选；勾选键为单据 id |
| 列宽拖拽 | 支持 |
| 固定列 | 采购退货出库单号列左固定；操作列右固定 |
| 分页 | 默认每页 20 条，支持 20/50/100 |
| 空状态 | 展示「暂无数据」插图 + 文字说明 |

### 5.2 列定义（从左到右）

| 列名 | 内容说明 | 宽度 | 固定 |
| :--- | :--- | :--- | :--- |
| 勾选列 | 复选框 | 48px | 左固定 |
| 采购退货出库单号 | `PRO{日期}-{序号}`，可点击进入详情页，链接色展示 | 190px | 左固定 |
| 出库状态 | 状态 Tag | 100px | - |
| 来源退货单号 | 可点击进入关联 PR 详情 | 180px | - |
| 来源入库单号 | 可点击进入关联 PI 详情 | 180px | - |
| 供应商 | `{Code} {Name}` | 200px | - |
| 出库仓库 | `{Code} {Name}` | 160px | - |
| 出库日期 | `YYYY-MM-DD` | 120px | - |
| 商品种数 | `N 种` | 80px | - |
| 出库总数量 | `N` | 100px | - |
| 出库总金额 | `¥1,234.56` | 130px | - |
| 最后修改时间 | `YYYY-MM-DD HH:mm:ss` | 170px | - |
| 操作 | 行内文字操作 | 140px | 右固定 |

### 5.3 状态 Tag 配色

| 状态 | 颜色语义 |
| :--- | :--- |
| 草稿 | 灰色 |
| 已确认 | 绿色 |
| 已作废 | 红色 |

### 5.4 行内操作

| 状态 | 展示的操作 |
| :--- | :--- |
| 草稿 | 查看 / 编辑 / 确认出库 / 作废 / 删除 |
| 已确认 | 查看 |
| 已作废 | 查看 |

不可用操作不渲染，不展示禁用灰色文字。

## 六、二次确认弹窗规格

| 触发操作 | 弹窗标题 | 弹窗内容 | 确认按钮文案 | 取消按钮文案 |
| :--- | :--- | :--- | :--- | :--- |
| 删除（草稿） | 确认删除 | 删除后不可恢复，该草稿退货出库单将从系统中永久移除，确认删除？ | 确认删除 | 取消 |
| 作废（草稿） | 确认作废 | 作废后不可恢复，单据将进入已作废状态，确认作废？ | 确认作废 | 取消 |
| 确认出库 | 确认出库 | 确认后将扣减现存库存、冲减供应商应付并生成库存流水，确认出库？ | 确认出库 | 取消 |

## 七、Mock 数据规格

### 7.1 状态覆盖要求

| 状态 | 最少条数 | 备注 |
| :--- | :--- | :--- |
| 草稿 | 2 | 覆盖可编辑和可确认 |
| 已确认 | 4 | 覆盖已生成 FL |
| 已作废 | 1 | 只读 |
| 总计 | >= 7 | 日期均使用 2026 年 |

### 7.2 数据要求

- 供应商展示格式：`S001 深圳强盛电子`、`S002 东莞恒科配件`。
- 出库仓库展示格式：`WH001 民房一号仓`。
- PRO 单号格式：`PRO20260707-0001`，来源 PR 格式：`PR20260707-0001`。
- 出库总金额范围：100.00 至 50,000.00。
