## 一、文档定位

本文件是采购退货单（PR）所有字段的唯一定义来源。下游业务规则、流程推演和 Demo PRD 只引用字段名称，不重复定义字段枚举、必填性和计算规则。

**引用规则**：
- 采购退货单继承采购入库单（PI）字段时，标注「继承PI」，不重复改写 PI 字段本体。
- 采购退货出库单（PRO）引用 PR 时，应继承本文件字段口径。
- 字段定义变化时，优先修改本文件，再同步重生成下游文档。

## 六、展示与引用规则

- PR 被 PRO 引用时，展示格式为采购退货单号，如 `PR20260707-0001`。
- 列表优先展示列：采购退货单号、来源入库单号、供应商、退货仓库、退货日期、退货状态、商品种数、退货总金额、最后修改时间。
- 所有继承 PI 的供应商、仓库、商品、单价、税率字段均为快照只读字段，不允许在 PR 修改。
- PR 确认后所有字段锁定为只读；如需实际扣减库存和冲减应付，必须由已确认 PR 下推生成 PRO 并在 PRO 确认出库时执行。
- 数量字段按整数展示；金额字段按 `¥1,234.56` 展示。
