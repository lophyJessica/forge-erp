# 采购退货单_Demo_详情页

> **状态**：已补齐
> **角色**：Demo·详情页　|　类型：业务单据

---

## 一、页面概述

| 项目 | 内容 |
| :--- | :--- |
| 页面名称 | 采购退货单详情 |
| 页面目标 | 只读展示采购退货申请全部信息，提供状态相关操作入口和上下游追溯 |
| 页面路径 | `/purchase/returns/{id}` |
| 骨架结构 | 页头（标题+状态+操作）→ 来源信息卡片 → 退货信息卡片 → 退货明细卡片 → 关联单据卡片 → 制单信息卡片 |

---

## 二、页头

| 元素 | 规格 |
| :--- | :--- |
| 左侧标题 | `采购退货单 {采购退货单号}` |
| 左侧副标题 | 状态 Tag |
| 右侧操作区 | 按状态动态展示按钮 |
| 返回列表 | 固定放在操作组最左侧 |

### 2.1 页头操作按钮矩阵

| 按钮 | 草稿 | 已确认 | 已作废 |

| 返回列表 | 是 | 是 | 是 |
| 编辑 | 是 | 否 | 否 |
| 删除 | 是 | 否 | 否 |
| 作废 | 是 | 否 | 否 |
| 确认退货 | 是 | 否 | 否 |
| 下推退货出库 | 否 | 是（条件） | 否 |
| 导出 | 是 | 是 | 是 |

不满足状态或权限时不渲染按钮。

## 三、来源信息卡片

卡片标题：**来源信息**。全部字段为只读展示。

| 字段 | 展示格式 | 说明 |
| :--- | :--- | :--- |
| 来源入库单号 | 链接文本，如 `PI20260704-0001` | 点击跳转 PI 详情 |
| 来源采购单号 | 链接文本，如 `PO20260704-0001` | 点击跳转 PO 详情 |
| 供应商 | `{Code} {Name}` | 继承 PI |
| 退货仓库 | `{Code} {Name}` | 继承 PI 入库仓库 |

---

## 四、退货信息卡片

卡片标题：**退货信息**。4 列栅格，全部为 `ReadonlyText`。

| 字段 | 展示格式 |
| :--- | :--- |
| 采购退货单号 | `PR20260707-0001` |
| 退货状态 | 状态 Tag |
| 退货日期 | `YYYY-MM-DD` |
| 退货原因 | 直接展示；无值展示 `-` |
| 退货备注 | 直接展示；无值展示 `-` |

---

## 五、退货明细卡片

卡片标题：**退货明细**。标题右侧展示 `共 N 种商品 | 总数量 N 件 | 总金额 ¥N`。

### 5.1 明细表格

| 列名 | 内容说明 | 宽度 |
| :--- | :--- | :--- |
| 序号 | 1、2、3 | 60px |
| 商品编码 | 直接展示 | 120px |
| 商品名称 | 直接展示 | 180px |
| 商品条码 | 无值展示 `-` | 140px |
| 规格型号 | 无值展示 `-` | 120px |
| 单位 | 直接展示 | 80px |
| 原入库数量 | 整数 | 110px |
| 已退货申请数量 | 整数 | 130px |
| 可退货数量 | 整数 | 110px |
| 本次退货数量 | 整数 | 120px |

| 退货金额（含税） | `¥N,NNN.NN` | 140px |
| 行备注 | 无值展示 `-` | 160px |

### 5.2 汇总行

| 汇总项 | 展示内容 |
| :--- | :--- |
| 商品种数 | `共 N 种商品` |

## 六、关联单据卡片

卡片标题：**关联退货出库单**。

| 展示条件 | 展示方式 |
| :--- | :--- |
| 草稿 | 不展示该卡片 |
| 已确认且暂无 PRO | 展示空态：`尚未生成采购退货出库单` |
| 已确认且已有 PRO | 展示 PRO 表格 |
| 已作废 | 不展示该卡片 |

### 6.1 PRO 表格列

| 列名 | 内容说明 |
| :--- | :--- |
| 退货出库单号 | 链接文本，点击跳转 PRO 详情 |
| 出库仓库 | `{Code} {Name}` |
| 出库日期 | `YYYY-MM-DD`；无值展示 `-` |
| 出库数量 | 整数 |
| 状态 | PRO 当前状态 Tag |

## 八、二次确认弹窗规格

| 触发操作 | 弹窗标题 | 弹窗内容 | 确认按钮 | 取消按钮 |
| :--- | :--- | :--- | :--- | :--- |
| 删除 | 确认删除 | 删除后不可恢复，该草稿退货单将从系统中永久移除，确认删除？ | 确认删除 | 取消 |
| 作废 | 确认作废 | 作废后该退货单将永久失效，确认作废？ | 确认作废 | 取消 |
| 确认退货 | 确认退货 | 确认后退货申请将锁定，并可下推采购退货出库单，确认继续？ | 确认退货 | 取消 |

---

## 九、空值占位和展示规则

| 规则 | 说明 |
| :--- | :--- |
| 空值 | 统一展示 `-` |
| 主数据引用 | 统一展示 `{Code} {Name}` |
| 金额 | 保留 2 位小数，千分位展示 |
| 数量 | 整数展示 |
| 日期时间 | 日期 `YYYY-MM-DD`；时间戳 `YYYY-MM-DD HH:mm:ss` |
| 只读展示 | 使用只读文本，不使用 disabled 输入框 |

## 十、反馈文案

| 操作 | 反馈类型 | 文案 |
| :--- | :--- | :--- |
| 删除成功 | Toast + 返回列表 | 采购退货单已删除 |
| 作废成功 | Toast + 页面刷新 | 采购退货单已作废 |
| 确认退货成功 | Toast + 页面刷新 | 采购退货单已确认 |
| 下推 PRO 成功 | Toast + 跳转 | 已生成采购退货出库单 |
| 状态不可编辑 | Toast | 当前状态不可编辑 |
