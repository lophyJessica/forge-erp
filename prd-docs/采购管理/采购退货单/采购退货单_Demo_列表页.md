## 一、页面概述

| 项目 | 内容 |
| :--- | :--- |
| 页面名称 | 采购退货单列表 |
| 页面目标 | 展示采购退货申请，支持按来源入库单、供应商、仓库、状态筛选，并提供查看、编辑、确认、下推 PRO 等入口 |

## 二、状态 Tab 栏

| Tab 标签 | 过滤逻辑 |
| :--- | :--- |
| 全部 | 展示所有状态的采购退货单 |
| 草稿 | 仅展示草稿 |
| 已确认 | 仅展示已确认 |
| 已作废 | 仅展示已作废 |

**交互规则**：Tab 右侧展示计数；切换 Tab 后刷新列表并重置到第 1 页，不清空其他查询条件；切换到具体状态 Tab 后，查询区隐藏「退货状态」筛选。

---

## 三、查询区

### 3.1 布局说明

- 桌面端 6 列栅格，默认展示一行条件；展开后展示更多条件。
- 右侧操作组：`展开/收起` → `重置` → `搜索`。

### 3.2 查询条件

| 字段 | 控件类型 | 规格说明 |
| :--- | :--- | :--- |
| 采购退货单号 | `BatchSearchInput` | 支持换行或英文逗号分隔多值，精确匹配；最多 1000 行 |
| 来源入库单号 | `BatchSearchInput` | 支持多 PI 单号精确匹配 |
| 来源采购单号 | `BatchSearchInput` | 支持多 PO 单号精确匹配 |
| 供应商 | `SelectWithSearch` | 展示 `{Code} {Name}`；支持编码或名称搜索；可清空 |
| 退货仓库 | `SelectWithSearch` | 展示 `{Code} {Name}`；支持编码或名称搜索；可清空 |
| 退货状态 | `Select` | 枚举见字段清单；仅在「全部」Tab 下展示 |
| 退货日期 | `DateRange` | 精确到日；可只选一端；含今天/本周/本月快捷项 |
| 最后修改时间 | `DateRange` | 展开后展示 |

## 四、工具条

| 按钮 | 位置 | 状态 | 说明 |
| :--- | :--- | :--- | :--- |
| 发起退货 | 左侧 | 常驻 | 打开“选择采购入库单”弹窗；只允许选择已确认且有可退货数量的 PI |

## 五、表格

### 5.1 表格行为

| 行为 | 规格 |
| :--- | :--- |
| 行勾选 | 支持，全选/单选；勾选键为单据 id |
| 列宽拖拽 | 支持 |
| 固定列 | 采购退货单号左固定；操作列右固定 |
| 分页 | 默认每页 20 条，支持 20/50/100 |
| 空状态 | 展示「暂无数据」插图 + 文案「暂无采购退货单」 |

### 5.2 列定义

| 列名 | 内容说明 | 宽度 | 固定 |
| :--- | :--- | :--- | :--- |
| 勾选列 | 复选框 | 48px | 左固定 |
| 采购退货单号 | 可点击进入详情页，链接色展示 | 180px | 左固定 |
| 退货状态 | 状态 Tag | 100px | - |
| 来源入库单号 | 可点击跳转 PI 详情 | 180px | - |
| 来源采购单号 | 可点击跳转 PO 详情 | 180px | - |
| 供应商 | `{Code} {Name}` | 200px | - |
| 退货仓库 | `{Code} {Name}` | 160px | - |
| 退货日期 | `YYYY-MM-DD` | 120px | - |
| 商品种数 | `N 种` | 80px | - |
| 退货总金额 | `¥1,234.56` | 130px | - |
| 最后修改时间 | `YYYY-MM-DD HH:mm:ss` | 170px | - |
| 操作 | 行内文字操作 | 180px | 右固定 |

### 5.3 状态 Tag 配色

| 状态 | 颜色语义 |
| :--- | :--- |
| 草稿 | 灰色 |
| 已确认 | 绿色 |
| 已作废 | 红色 |

### 5.4 行内操作

| 状态 | 展示的操作 |
| :--- | :--- |
| 草稿 | 查看 / 编辑 / 确认退货 / 删除 / 作废 |
| 已确认 | 查看 / 下推退货出库 |
| 已作废 | 查看 |

不渲染 disabled 操作；没有权限或状态不满足时直接隐藏入口。

## 六、选择采购入库单弹窗

| 区域 | 规格 |
| :--- | :--- |
| 标题 | `选择采购入库单` |
| 查询条件 | 入库单号、供应商、入库仓库、入库日期 |
| 可选范围 | 已确认 PI，且至少一行可退货数量大于 0 |
| 表格列 | 入库单号、供应商、入库仓库、入库日期、商品种数、入库总金额、可退货总数量 |
| 确认按钮 | `发起退货` |
| 行选择 | 单选 |

确认后跳转至 `/purchase/returns/new?sourcePiId={piId}`，并预填 PR 新增页。

## 七、二次确认弹窗规格

| 触发操作 | 弹窗标题 | 弹窗内容 | 确认按钮文案 | 取消按钮 |
| :--- | :--- | :--- | :--- | :--- |
| 删除 | 确认删除 | 删除后不可恢复，该草稿退货单将从系统中永久移除，确认删除？ | 确认删除 | 取消 |
| 作废 | 确认作废 | 作废后该退货单将永久失效，确认作废？ | 确认作废 | 取消 |
| 确认退货 | 确认退货 | 确认后退货申请将锁定，并可下推采购退货出库单，确认继续？ | 确认退货 | 取消 |

## 八、Mock 数据规格

| 状态 | 最少条数 | 备注 |
| :--- | :---: | :--- |
| 草稿 | 3 | 覆盖可编辑、可删除、可作废 |
| 已确认 | 4 | 覆盖可下推 PRO |
| 已作废 | 2 | 只读 |
| 总计 | >= 9 | 日期均使用 2026 年 |

**示例要求**：单号使用 `PR20260707-0001`、`PI20260704-0001`、`PO20260704-0001`；供应商可用 `S001 深圳强盛电子`、`S002 东莞恒科配件`；金额范围 100.00 至 50,000.00。
