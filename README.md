# 强盛进销存 —— 第 4 节演示项目

> 本目录是课程第 4 节「从地基到产出预览」的课堂演示项目，配合讲稿使用。

## 目录角色

| 目录 | 角色 | 说明 |
|---|---|---|
| `context/` | **AI 上下文源头** | 9 个文件，其中 04/06/07 为骨架（课堂由 AI 填充），其余为预填 |
| `templates/` | **格式规范与模板** | AI 生成 PRD / 字段清单 / 状态机 / Demo PRD / 前端页面时的输出标准 |
| `prd-docs/` | **PRD 产出物** | 采购订单已有完整 PRD（交叉验证参照物）；采购入库单/退货单/退货出库单为空（课堂由 AI 生成） |
| `front-prototype/` | **前端原型** | 课堂演示从 PRD 生成页面骨架，目前为空 |
| `prompts/` | **通用提示词库** | 8 个可复用的提示词模板，供学员课后使用 |
| `演示提示词/` | **课堂演示脚本** | 7 步演示提示词 + 讲稿，讲师上课直接复制使用 |
| `drafts/` | **临时草稿** | 空，课堂演示展示用途 |
| `references/` | **参考资料** | 原项目背景、术语字典、角色权限等历史资料 |

## 课堂演示顺序

1. 展示项目结构，圈出 🟠 文件 + 空目录
2. AI 填充 context/04（功能模块）
3. AI 填充 context/06（业务场景）
4. AI 填充 context/07（业务约束）
5. AI 粗跑采购入库单 PRD 初稿
6. AI 粗跑采购订单前端页面骨架
7. AI 自查 + 提示词归档

详见 `演示提示词/00-演示路线总览.md`。

## 注意事项

- 本节定位为**全流程预览课**，所有产出均为"初稿/骨架/粗版"，不追求可评审质量
- 各环节的精细做法在后续课程中拆开精讲
- 演示前建议实际运行一次，确认 AI 产出在预期范围内
